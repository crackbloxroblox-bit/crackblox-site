How to publish on Netlify:
1. Zip the crackblox-website folder (or keep as folder).
2. Visit https://app.netlify.com/drop
3. Drag & drop the crackblox-website folder (NOT the contents). Netlify will deploy.
4. Visit YOURSITE.netlify.app/admin to log in and edit (setup identity/gate)
Note: To enable Netlify CMS editing you must enable Git Gateway or Identity in Netlify settings (Netlify docs).
